# Ben Communication Software
An application made from python in about a week of tinkering and some help from ChatGPT.


This was able to give my brother, a nonverbal quadriplegic who only has access to left and right head switches (space and return) freedom in his daily life in a situation where he otherwise wouldn't have any.
